Login

1 Install Device ID (mirror) on your Android device. This app will show necessary information for you to use on APK Downloader.
2 Log in to APK Downloader using the information showed in Device ID app.

Enter your email showed in the Device ID app. Do not log in with another email, or it won't work.

2-step verification is not supported. If you're using it, please use Application-specific password instead.

This password is used once to retrieve an authentication token and is not stored.
Note: If you entered correct email and password but still couldn't log in, please turn on Access for less secure apps setting and try again.

Make sure you entered it correctly, or you won't be able to download APKs.
